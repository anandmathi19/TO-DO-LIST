To-Do List Website

A fully functional **To-Do List web application** built using **Flask**, **Python**, and **HTML/CSS**.  
Easily manage tasks by adding, completing, deleting, and filtering them.

---

## Features

- Add new tasks  
- Mark tasks as completed / pending  
- Delete individual tasks  
- Clear all completed tasks at once  
- Filter tasks: **All**, **Pending**, **Completed**  
- Live task counter showing total and completed tasks  
- Simple, responsive, and user-friendly design  

---

## Technologies Used

- **Python 3**  
- **Flask** – Web framework  
- **HTML / CSS** – Frontend  
- **Jinja2** – Templating engine
